Nessa aula criamos um sistema mínimo de backend com CRUD e uma arquitetura services/controllers

Utilizamos as seguinte tecnologias
- Express
- Prisma
- SQLite
- Typescript

Para instalar as dependências.
- npm i

Criar o arquivo .env na raiz do projeto:
- Adicionar a variável DATABASE_URL="file:./app.db"

Para gerar o banco de dados:
- npx prisma generate

Para rodar o projeto:
- npm run dev

Para abrir o prisma studio no navegador:
- npx prisma studio
